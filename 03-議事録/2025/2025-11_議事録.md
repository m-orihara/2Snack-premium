## 2025-11-07 プレミアムライン販売戦略会議

**日時**: 2025-11-07
**場所**: オンライン
**参加者**: 平慎司、社長
**記録者**: 平慎司

### アジェンダ

1. プレミアムライン新製品（チョコ、チーズケーキラスク）のローンチスケジュール確認
2. 価格設定と利益率の検証
3. 販売チャネル戦略の見直し

### 議論内容

#### 1. 製品ローンチスケジュール

**チョコレート製品**

- ローンチ予定: 3月（最短）
- 配合フィックス・パッケージデザインへの着手タイミングを検討中
- 価格・販路が固まってから実機テストに入るか、並行して進めるかを判断する必要がある

**チーズケーキラスク**

- ローンチ予定: 9月頃（来年秋）
- 理由: 6ヶ月賞味期限を確保するため、最短でも秋ローンチとなる
- 現在、配合・パッケージデザインの準備段階

**議論のポイント**

- 元々ファミマの棚入れを想定して急いでいたが、ファミマの状況が想定通りに進まない可能性がある
- ファミマ依存のスケジュールではなく、通常のプレミアムリテール展開を前提に進める方向

#### 2. 価格設定・利益率

**チョコレート製品**

| 項目         | 内容                                        |
| ------------ | ------------------------------------------- |
| 希望小売価格 | 378円（税込）                               |
| 内容量       | **35g**（40gではなく35gで確定）       |
| 卸掛け率     | 氷30%、おろし15%                            |
| パッケージ   | チャック付きスタンドパウチ（17.5円/個）     |
| 物流費       | 夏場は低温保管・クール配送前提で算出        |
| 営業利益率   | 約20%前後（ミニマムロット500kg時）          |
| 生産ロット   | ミニマム500kg、倍量でコストダウン可能性あり |

**課題・検討事項**

- コンビニ掛け率40-43%の場合、利益率が圧迫される
- 希望小売価格を設定し、チャネル別に卸価格を調整する方式で対応
- 加重平均で25%程度の営業利益率を目標とする
- 経済ロット（500kg vs 1000kg）での価格差を早急に確認する必要がある
- 市場比較: 既存製品は40g/320円程度で展開していた（期間限定）
- 差別化要素: 健康価値やパッケージングでの差別化が重要

**チーズケーキラスク（3枚入り）**

| フレーバー   | 希望小売価格  | 中身原価 | 営業利益率 |
| ------------ | ------------- | -------- | ---------- |
| チーズケーキ | 594円（税込） | 194円    | 16%        |
| チョコ       | 594円（税込） | [高め]   | 14%        |
| 抹茶         | 594円（税込） | [最高]   | 7%         |

**課題・検討事項**

- 抹茶は利益率が低すぎる（7%）ため、ビジネスとして成立しづらい
- 抹茶は遮光パッケージ（銀蒸着）必須でコスト増
- 抹茶原料の種類・グレードによる価格差を確認する必要がある
- 経済ロット: 100万円分＝約1,300袋/月
- 市場では3枚入り高級ラスクが一定の価格帯で流通しており、価格的な違和感は少ない見込み
- 戦略: まずチーズケーキ・チョコの2種で市場投入し、反応を見てから抹茶を検討する選択肢もある
- 3種展開の方が売場でのインパクト・存在感は強い
- 3製品同時発注での価格交渉余地を確認する

#### 3. 販売チャネル戦略

**現状の想定チャネル**

- ナチュラルローソン（重要度: ★高）
- プラザ（重要度: ★高）
- ファミリーマート（難易度高い、不確実性大）

**議論された課題**

- ファミマ一択は危険 → 1か8かのような状況になりかねない
- ナチュラルローソン・プラザへのウェイトが大きく、ここが取れれば回る見込み
- しかし、1本に絞るとリスクが高い

**新しい方向性**

- セカンド・サードオプションを用意する
- プレミアムリテールを軸に、複数パターンのシミュレーションを作成
  - パターン1: ナチュラルローソン・プラザが入った場合
  - パターン2: どちらかが入らなかった場合の代替チャネル
  - パターン3: 代替チャネルでの展開
- ドラッグストアは優先度低い（ブランドイメージ的に合わない）
- コンビニの一部店舗（セブンイレブン、ローソンなど）
- 成城石井、いかりスーパー、阪急オアシスなど高級スーパー
- スーパーの洋菓子・和菓子コーナー（300-400円の菓子が普通に並んでいる）
- 駅ナカや銀座などプレミアムエリアでの販売も検討

**リスク管理の考え方**

- 最悪ケースで在庫が数十万円規模なら許容範囲内
- 全店舗展開ではなく、一部店舗での展開を前提とする
- 在庫リスクを最小化しながら、複数チャネルで回せる体制を構築

**プラザとの商談**

- 村井さんが今月正式な商談予定
- 2Snackの他製品と合わせて提案する方向

### 決定事項 ★重要

- [X] チョコレート製品は**35g**で進める（40gは不採用）
- [X] チーズケーキラスクは**9月ローンチ**目標で進行
- [X] 販売チャネルは**セカンド・サードオプションを必ず用意**する
- [X] 価格は**希望小売価格を設定し、チャネル別に卸価格を調整**する方式
- [X] 抹茶フレーバーは経済性確認後に判断（**チーズケーキ・チョコを優先**）
- [X] ファミマ依存ではなく、**プレミアムリテール中心**の戦略で進める
- [X] リスク許容範囲: **最悪ケースで在庫数十万円規模**

## 2025-11-27 商品開発ディスカッション（含浸チョコ・ケーキラスク）

**日時**: 2025-11-27
**場所**: オンライン
**参加者**: 清信奏江、二本木ちひろ（開発担当）、縄巻蘭
**記録者**: 清信奏江

### アジェンダ

1. 含浸チョコ（おかき）とケーキラスクの商品説明（開発者視点）
2. 製造工程と技術的特徴の確認
3. 原材料・素材のこだわりポイント
4. 商品ネーミング案の検討

### 議論内容

#### 1. 商品コンセプト・開発背景

**基本コンセプト**

- 「欲しいものはいつも2つ」という開発思想
- 意外性のある2つの組み合わせ：
  - 和×洋（おかき×チョコレート、ラスク×チョコレート）
  - 味の組み合わせ（抹茶×塩、オレンジ×ビターチョコ）

**開発の経緯**

- 当初はラスクでプレミアム商品を検討
- ガトーフェスタ・ハラダのような上にチョコをかけるだけでは面白くない
- プラスアルファの要素を模索：生地に練り込む、キャラメルコーティング、バター漬けなど
- 実現可能性（賞味期限、温度管理）の壁に直面
- シミチョコ（コーンパフ等）を食べた際の発想：「安い感じするけどめっちゃ美味しい」
- 社長も「シミチョコ好きなんだよね」との発言
- 北海道の高級ドライいちごホワイトチョコの成功事例を参考
- 「これをプレミアムにするには？」→ 素材をプレミアムにする発想
- 和×洋の組み合わせで、おかき×チョコレートに到達

**シリーズ展開の可能性**

- 吸わせる技術は様々な素材に応用可能
- フリーズドライ野菜（ブロッコリー、トウモロコシ等）への展開も構想中
- 実例：無印良品のドライみかん・りんご、カルディのキウイフルーツなど

#### 2. 製造工程と技術的特徴

**独自配合の植物性チョコレート**

- うちのためだけに作った独自配合の植物性チョコレート
- よくあるチョコレートがけ・コーティングとは差別化
- **吸わせる専用のチョコレート**として開発

**チョコレート配合の工夫**

- 油脂を多く配合してサラサラの状態に（ドロっとしたチョコでは吸わない）
- 水を入れると分離するため、油脂で粘度を調整

**吸わせる工程**

- 長い時間をかけてじっくり染み込ませる（プレミアム製法）
- 具体的な時間：10時間程度（要確認）※一般的な基準との比較で訴求力を検討中
- 手作業での特別な工程はなし（工場での機械製造）

**乾燥工程**

- チョコレートを吸わせた後、乾燥させる
- サクッとパリッとした食感を維持
- 濡れ煎餅のようにベタベタにならない

**チョコレートの配合比率**

- おかき1に対してチョコレート2.4の比率
- チョコレートが倍以上入っており、濃厚な仕上がり

#### 3. 原材料・素材のこだわり

**おかき（含浸チョコ）**

- 米：もち米使用
- 産地：秋田県産、北海道産
- ※訴求力について要検討

**ラスク**

- バゲットを仕入れて使用（パンから焼いているが、工場で生地から作っているわけではない）
- 植物性のバゲット使用
- 特別な差別化要素は少ない

**チョコレート**

- クーベルチュール使用とは言えない
- オリジナル配合の吸わせる専用チョコレート

**抹茶（要確認事項）**

- 京都産の使用可否：現在コスト次第で調整中
- チョコレート屋さんが使用している抹茶の価格開示待ち
- 二本木さん手配の抹茶の方が安ければ変更提案予定
- **確認時期**：12月頭頃
- ※「京都宇治産」「一番摘み抹茶」等の訴求ができれば強い差別化要素となる

**オレンジフレーバー**

- 何度も試作を重ねて調整
- 当初はガツンとオレンジ感を出していたが、上品でプレミアム感を出すためにあえて控えめに
- 袋を開けた時にふわっと香る程度に調整
- オレンジピールの使用有無：要確認
- オレンジフレーバー使用

**抹茶×塩の配合**

- 何度も試作を重ねて甘さと塩味のバランスを調整
- ※塩味の強さについて要再確認

#### 4. 商品ネーミング案の検討

**現状の検討方向性**

- 3要素（染み・チョコ・おかき/ラスク）が基本
- プラスアルファすると長くなる課題

**おかき製品の候補案**

| 提案者 | 案                              | 備考                                 |
| ------ | ------------------------------- | ------------------------------------ |
| -      | 染みチョコおかき                | シンプル案                           |
| -      | 染み込みチョコおかき            | 「染み込み」表現                     |
| -      | 濃厚/リッチ + 染みチョコおかき  | プレミアム感追加                     |
| 二本木 | 染み込むラスク ビターオランジェ | 「染み込む」動詞表現、フレーバー明記 |
| 二本木 | 米ショコラ 抹茶と淡塩           | おかき=お米の訴求、繊細な印象        |

**ラスク製品の候補案**

| 検討ポイント           | 内容                                                          |
| ---------------------- | ------------------------------------------------------------- |
| 「ラスク」の印象       | ガトーフェスタ・ハラダのような高級ラスクのイメージが一般的    |
| チープな印象リスク     | 食パンの耳を揚げて砂糖をつけたようなラスクのイメージもある    |
| 対策                   | プレミアム/リッチ等の形容詞を追加してパッケージで高級感を演出 |
| ケーキラスクとの統一性 | 今後出すケーキラスクも「ラスク」表現を使う予定                |
| 代替案検討             | クリスピーケーキ等の表現も検討可能                            |

**ネーミングの方向性まとめ**

- フレーバー要素は必須（シリーズ化を見据えて）
- 絵でフレーバーを表現するか、言葉で表現するか
- リッチ感・プレミアム感を出す形容詞の追加
- おかき：「米」要素の訴求も検討
- ラスク：チープ感を避けるための工夫が必要

#### 5. その他の論点

**開発で難しかったポイント**

- 「これとこれを掛け合わせたい」というコンセプトを超えてしまえば、特に問題なし
- 作れる工場を探して、フレーバーを依頼しただけ
- （PR的には）オレンジフレーバーと抹茶の甘さ・塩味のバランス調整を何度も試行錯誤

**ザクザク食感の維持**

- 特別な製法ではない
- チョコレートの配合（油脂・ココアバター）と乾燥工程、素材（おかき・ラスク）自体のカリッとした特性に由来
