# 04-価格

商品ライン別の価格情報（基本価格 + 企業別提出価格）を管理するフォルダです。

## フォルダ構成

```
04-価格/
├── README.md
├── ケーキラスク/
│   └── 価格表.md    … 基本価格（ベース） + 企業別提出価格一覧
└── 含浸チョコ/
    └── 価格表.md    … 基本価格（ベース） + 企業別提出価格一覧
```

## 計算前提

- 小売粗利率：30%
- 取引形態：卸経由（TWO → 卸 → 小売）
- 未収：基本計算には含めない

## 企業別一覧の列定義（全21列）

| # | 列名 | 説明 |
|---|------|------|
| 1 | 企業名 | 取引先企業名 |
| 2 | 商品名 | 提出した商品名（SKU） |
| 3 | 販売価格 | 消費者向け売価（税抜） |
| 4 | 小売粗利額 | 小売店の粗利金額 |
| 5 | 小売粗利率 | 小売店の粗利率 |
| 6 | 小売への納価 | 卸→小売への納品価格 |
| 7 | 卸粗利額 | 卸の粗利金額 |
| 8 | 卸粗利率 | 卸の粗利率 |
| 9 | 仕切り値 | TWO→卸への価格 |
| 10 | 未収 | 未収金額 |
| 11 | ネット価格 | 未収控除後の価格 |
| 12 | 卸掛け率 | 仕切り値 ÷ 販売価格 |
| 13 | TWO粗利額 | TWOの粗利金額（ネット価格 − 原価） |
| 14 | TWO粗利率 | TWOの粗利率 |
| 15 | 原価 | 商品原価 |
| 16 | 下代に対する原価率 | 原価 ÷ 仕切り値 |
| 17 | 販管費合計 | 送料 + その他 |
| 18 | 販管費率 | 販管費合計 ÷ ネット価格 |
| 19 | 送料 | 物流費 |
| 20 | TWO営業利益額 | TWO粗利額 − 販管費合計 |
| 21 | TWO営業利益率 | TWO営業利益額 ÷ ネット価格 |
