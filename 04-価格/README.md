# 04-価格

商品ライン別の価格情報（基本価格 + 企業別提出価格）を管理するフォルダです。

## フォルダ構成

```
04-価格/
├── README.md
├── ケーキラスク/
│   └── 価格表.md    … 基本価格（ベース） + 企業別提出価格一覧
└── 含浸チョコ/
    └── 価格表.md    … 基本価格（ベース） + 企業別提出価格一覧
```

## 企業別一覧の列定義

| 列名 | 説明 |
|------|------|
| 企業名 | 取引先企業名 |
| 商品名 | 提出した商品名（SKU） |
| 販売価格 | 消費者向け売価（税抜） |
| 小売粗利額 | 小売店の粗利金額 |
| 小売粗利率 | 小売店の粗利率 |
| 小売への納価 | 卸→小売への納品価格 |
| 卸粗利額 | 卸の粗利金額 |
| 卸粗利率 | 卸の粗利率 |
| 仕切り値 | TWO→卸への価格 |
| 未収 | 未収金額 |
| ネット価格 | 未収控除後のネット価格 |
| 卸掛け率 | 卸掛け率 |
| 卸粗利額 | 卸粗利金額 |
| 卸粗利率 | 卸粗利率 |
| 原価 | 商品原価 |
| 原価率 | 原価率 |
| 下代に対する原価率 | 下代ベースの原価率 |
| 販管費合計 | 販管費の合計金額 |
| 販管費率 | 販管費率 |
| 送料 | 送料・物流費 |
| TWO営業利益額 | TWOの営業利益金額 |
| TWO営業利益率 | TWOの営業利益率 |
