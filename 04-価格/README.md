# 04-価格

商品ライン別の価格情報（基本価格 + 企業別提出価格）を管理するフォルダです。

## フォルダ構成

```
04-価格/
├── README.md
├── ケーキラスク/
│   └── 価格表.md    … 基本価格（ベース） + 企業別提出価格一覧
└── 含浸チョコ/
    └── 価格表.md    … 基本価格（ベース） + 企業別提出価格一覧
```

## 計算前提

- 小売粗利率：30%
- 取引形態：卸経由（TWO → 卸 → 小売）
- 未収：基本計算には含めない

## ステータス定義

| アイコン | ステータス | 意味 |
|----------|-----------|------|
| 🔵 | 確定 | 先方と合意済み |
| 🟡 | 社内確認中 | 社内で価格検討中 |
| 🟠 | 商談中 | 先方に提出・交渉中 |
| 🔴 | 不採用 | 先方から不採用の回答 |
| ⚪ | 未提出 | まだ提出していない |

1企業に複数パターン（値入れ変更案・バッファー計算等）がある場合は、同一企業名で複数行を記載。
