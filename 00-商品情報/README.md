# 00-商品情報

各商品の基本情報をまとめたフォルダです。
商品ごとにファイルを作成し、テンプレートに沿って情報を記載してください。

## ファイル一覧

| ファイル名 | 商品 |
|-----------|------|
| ケーキラスク_チーズケーキ.md | ケーキラスク チーズケーキ |
| ケーキラスク_チョコレート.md | ケーキラスク チョコレート |
| ケーキラスク_抹茶.md | ケーキラスク 抹茶 |
| 含浸チョコ_オレンジチョコレートラスク.md | 含浸チョコ オレンジチョコレートラスク |
| 含浸チョコ_抹茶塩おかき.md | 含浸チョコ 抹茶塩おかき |
| テンプレート.md | 新商品追加時のテンプレート |

## 記載項目一覧

| カテゴリ | 項目 |
|---------|------|
| 基本情報 | 商品名、発売日、企画 |
| 販売・製造 | ケース入り数、販売者、製造者、製造者住所 |
| コード類 | 商品コード、JANコード、ケースJAN、ITFコード |
| 品質管理 | 賞味期限、流通温度帯 |
| 商品詳細 | 商品詳細、保険材料の配合、商品画像、アレルゲン情報、栄養成分表示 |
| 価格情報 | 売価、卸納価、商品原価、粗利、物流費、営業利益 |
| 物流・発注 | リードタイム、製造リードタイム、発注ロット、包材ロット、受発注リードタイム、受発注ロット、受発注スケジュール |
| サイズ・重量 | 重量、包材サイズ、ケースサイズ |
