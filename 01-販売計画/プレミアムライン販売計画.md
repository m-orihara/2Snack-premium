**PLAZA先行ケースを試算に含める理由:**

- PLAZAはお菓子カテゴリおよび2foodsブランドとの親和性が過去プロダクトの実績から高い
- 先行販売を実施した際の日販上昇率が高い実績がある
- 最低ロットを確実に回していくうえでPLAZAは重要な企業
- 先行販売の提案可能性を試算上で検証しておく必要がある

### 計算式

**ケーキラスク:**

```
売上/月 = 配荷店舗数 × 日販 × 30日 × 納価(315.4円)
数量/月 = 配荷店舗数 × 日販 × 30日
ロットとの差 = 数量/3か月 - 2,500pc
ロス金額 = ロットとの差(マイナス時) × 原価(3SKU平均222.6円)
粗利 = 数量 × チーズケーキ粗利(83.1円) ※最低粗利SKUで試算
営業利益 = 数量 × チーズケーキ営業利益(63.0円) ※最低粗利SKUで試算
```

**含浸チョコ:**

```
売上/月 = 配荷店舗数 × 日販 × 30日 × 納価(217.3円)
数量/月 = 配荷店舗数 × 日販 × 30日
ロットとの差 = 数量/4か月 - 15,000pc
ロス金額 = ロットとの差(マイナス時) × 原価(162.71円)
粗利 = 数量 × 粗利単価(54.59円)
営業利益 = 数量 × 営業利益単価(32.09円)
```

## PLAZA先行販売の効果分析（含浸チョコ）

### 検討背景

PLAZA先行販売を行う場合、他企業への導入開始が遅れるため、販売数量への影響を分析。

### 前提条件

| 項目                  | 値                                |
| --------------------- | --------------------------------- |
| ロット期間            | 4か月（120日）                    |
| PLAZA先行期間         | 2週間（14日）                     |
| PLAZA店舗数           | 120店舗                           |
| PLAZA日販（先行あり） | 0.4                               |
| PLAZA日販（先行なし） | 0.2                               |
| 他企業の販売期間      | 先行あり: 106日 / 先行なし: 120日 |

**補足**:

- 先行販売期間中、他企業への営業は可能だが、導入開始は先行期間終了後
- PLAZAの日販0.4は、先行販売後も継続（2gummyの実績より）

### 計算結果

#### PLAZAの貢献

| パターン       | 計算式                  | 数量/4か月         |
| -------------- | ----------------------- | ------------------ |
| 先行販売あり   | 120店舗 × 0.4 × 120日 | **5,760pc**  |
| 先行販売なし   | 120店舗 × 0.2 × 120日 | **2,880pc**  |
| **差分** |                         | **+2,880pc** |

## 結論・懸念点

### 商品別リスク比較

| 項目                  | 含浸チョコ            | ケーキラスク           |
| --------------------- | --------------------- | ---------------------- |
| 最低ロット            | 15,000pc/4か月        | 2,500pc/3か月          |
| リアルケース          | ▲1,960（ロット割れ） | +7,010（余裕あり）     |
| コンサバケース        | ▲6,984（ロット割れ） | +3,320（余裕あり）     |
| 最悪ケース（3社失注） | ▲8,860               | +1,835（ロットクリア） |
| リスク                | 高い                  | 低い                   |

### ケーキラスクの結論

- **全ケース・全失注パターンでロットクリア** - ロット2,500pcに対して十分な余裕
- **最悪ケース（PLAZA・NL・カフェランテ全失注）でも+1,835pc**
- リスクは極めて低い

### 含浸チョコの結論

- **リアルケース + EC** の販売計画で、ロットクリア可能（卸13,040 + EC2,500 = 15,540pc、ロット差+540）
- **PLAZA先行販売 + EC** で、さらに余裕を確保（卸14,735 + EC2,500 = 17,235pc、ロット差+2,235）
  - ※卸のみではロット割れ（▲265pc）のため、EC販売の確保が必須

### 懸念点

1. **含浸チョコ**: 採用率の高さ - 全体のアタックリストから **約96%の企業に採用が必要**
2. **含浸チョコ**: 主要企業依存リスク - PLAZA、NLどちらも失注すると、ロス金額が大きい（PLAZA失注で▲787,516円）
3. 両商品を同時展開する場合、営業リソースの配分が必要

### 対策

- アタックリストの拡充（確度「無」の企業へのアプローチ）
- PLAZA先行販売の交渉を両商品で進める
- 含浸チョコよりも**ケーキラスクを優先展開**する選択肢もあり

### パターン3: 既存アタック先の配荷店舗数引き上げ

既にリレーションがある企業で、リアル店舗数→目標店舗数に近づける。

**主要企業のリアル→目標の乖離:**

| ロフト                   |   20   |  72  |      +52      |                      +234pc |
| ハンプティーダンプティー |   10   |  42  |      +32      |                      +144pc |
| ハンズ                   |   5   |  33  |      +28      |                      +126pc |

### パターン4: EC販売比率の引き上げ

棚落ちリスクがなく、NLの「ロングセラーになりにくい」問題と対照的なチャネル。

**施策例:**

- 自社ECの強化（2foodsブランドサイト）
- モール出店（Amazon、楽天等）
- SNS施策との連動（Instagram、X等でのプロモーション）

**数値インパクト（含浸チョコ）:**

| シナリオ |  EC月商 | 数量/月 | 現状比 | NL補填率 |
| -------- | ------: | ------: | :----: | :------: |
| 現状     |  50万円 |   625pc |   -   |    -    |
| 1.5倍    |  75万円 |   938pc | +313pc |   40%   |
| 2倍      | 100万円 | 1,250pc | +625pc |   80%   |

> ECは在庫管理が自社でコントロールしやすく、安定した消化チャネルとして機能する

---

### パターン5: 卸以外のB2Bチャネル開拓

小売配荷とは別軸で数量を積む。

**施策例:**

- **法人ギフト・ノベルティ**: プレミアム感のある商品はギフト適性が高い。季節需要（お中元・お歳暮・年末年始）との連動
- **ホテル・カフェ等の業務用卸**: 2foodsブランドと親和性のある施設への提案
- **百貨店催事・ポップアップ**: 短期集中でのロット消化。ブランド認知にも寄与

**数値インパクト（含浸チョコ・概算）:**

| チャネル           | 想定数量/月 | NL補填率 | 備考                         |
| ------------------ | ----------: | :------: | ---------------------------- |
| 法人ギフト         |  100〜300pc | 13〜38% | 季節変動大。9〜12月は需要期  |
| ホテル・カフェ     |   50〜150pc |  6〜19%  | 小規模だが継続性あり         |
| 催事・ポップアップ |  200〜500pc | 26〜64% | 短期集中。月平均に均すと変動 |

> 単独での全量補填は難しいが、他パターンとの合わせ技として有効。特にロット消化期間（4か月）の中で催事を1〜2回挟むことで数百pcの上積みが期待できる

### 補填シナリオまとめ

NL不在時に780pc/月を埋めるための組み合わせ例（現実的な順に記載）:

|   優先度   | 組み合わせ例                          | 内訳                                                                      |  合計補填量/月  |     補填率     |
| :---------: | ------------------------------------- | ------------------------------------------------------------------------- | :--------------: | :------------: |
| **1** | **新規開拓 + B2B + 店舗数拡大** | アタック先追加(+230pc) + 催事・ギフト(+300pc) + 主要3社店舗数拡大(+250pc) | **+780pc** | **100%** |
|      2      | 新規開拓 + 店舗数拡大 + EC強化        | アタック先追加(+230pc) + 主要3社店舗数拡大(+250pc) + EC1.5倍(+313pc)      |      +793pc      |      102%      |
|      3      | 日販向上 + 新規開拓 + B2B             | 日販0.18化(+360pc) + 確度「無」有力3社(+207pc) + 催事(+200pc)             |      +767pc      |      98%      |

> **優先度1のシナリオが最も現実的。** アタック先の追加（議事録でも指示済み）、B2Bチャネル（催事・法人ギフト）、既存先の配荷店舗数拡大の3軸で、NL分780pc/月の全量補填が可能。いずれも既に動き出せる施策であり、単一施策への依存を避けた分散型のリスクヘッジとなる。
