## 競合比較と価格戦略

### 目指すべきポジションの競合2商品

2Snack含浸チョコが棚で直接並ぶ可能性が高く、かつポジショニングとして目指すべき商品は以下の2つ。

| 項目                   | **BRUNO クリスピーブラウニー**                | **Olive Young ベーグルチップ**                  | **2Snack 含浸チョコ（現行想定）** |
| ---------------------- | --------------------------------------------------- | ----------------------------------------------------- | --------------------------------------- |
| **税込売価**     | 453円                                               | 397〜398円                                            | 430円（税抜398円）                      |
| **内容量**       | 60g                                                 | 55〜60g                                               | 35g                                     |
| **グラム単価**   | 7.6円/g                                             | 6.6〜7.2円/g                                          | 12.3円/g                                |
| **原産国**       | タイ                                                | 韓国                                                  | 日本（国内製造）                        |
| **訴求軸**       | グルテンフリー・無添加（ココナッツ粉/ヒマラヤ岩塩） | 低温熟成ベーグル2度焼き・揚げていない・ギルティフリー | 乳酸菌K-1配合・含浸技術・素材こだわり   |
| **販売チャネル** | PLAZA・Biople・成城石井等                           | PLAZA・ロフト等                                       | PLAZA・ロフト・アミング等（予定）       |
| **メーカー**     | 株式会社尾賀亀（輸入）                              | CJグループ（韓国）                                    | TWO（自社）                             |

### 分析

#### 1. 価格帯の整合性

3商品とも**税込400〜450円の価格帯**にあり、バラエティショップの棚で並んでも価格の違和感はない。ただし内容量が35gと競合（55〜60g）より小さいため、グラム単価では約1.7〜1.9倍高い。「含浸技術によるプレミアム製法」「乳酸菌K-1配合」で付加価値の説明が必要。

#### 2. チャネルの重なり

BRUNO・Olive Youngともに**PLAZA・ロフト等のバラエティショップが主戦場**で、2Snackのメイン戦場と完全に一致する。棚で直接競合する前提での価格設定が求められる。

#### 3. 健康訴求の棲み分け

| 商品             | 健康訴求の方向性                                                         |
| ---------------- | ------------------------------------------------------------------------ |
| BRUNO            | 「入っていないもの」の訴求（グルテンフリー・無添加）                     |
| Olive Young      | 「製法」の訴求（揚げていない・低温熟成）                                 |
| **2Snack** | **「入っているもの」の訴求**（乳酸菌K-1配合）+ Phase 2で機能性表示 |

→ 2Snackは「プラスの健康価値」を訴求できる唯一のポジション。機能性表示取得後はさらに差別化が明確になる。

### 価格設定シミュレーション（営業利益率20%目標）

**前提条件**: 商品原価162.71円、物流費22.5円、小売粗利率30%

| 卸粗利率 | 税抜売価 | 税込売価        | 卸納価  | 営業利益率 | 競合との位置づけ                               |
| -------- | -------- | --------------- | ------- | ---------- | ---------------------------------------------- |
| 22%      | 424円    | **458円** | 231.5円 | 20.0%      | BRUNO（453円）とほぼ同等。Olive Youngより+60円 |
| 18%      | 404円    | **436円** | 231.5円 | 20.0%      | 現行430円に近い。BRUNO/Olive Youngの中間       |
| 15%      | 389円    | **420円** | 231.5円 | 20.0%      | Olive Young（397円）に近い価格帯               |

> **考察**: 税込430〜460円の範囲であれば競合と同じ棚に並んでも価格の違和感はない。BRUNOの453円が上限の目安となり、これを超えると「同じ棚で一番高い」になるリスクがある。卸粗利18%（税込436円）は、現行想定からの微増でバイヤーにも説明しやすく、営業利益率20%も達成できるバランスの良い着地点。

|                          | Phase 1                                                 | Phase 2                                   |

| **機能性表示**     | なし                                                    | あり（肌のうるおい+整腸）                 |

## Phase 1（2026年9月〜）: 売り場展開

### 1. メイン戦場：バラエティショップのエンド・レジ前・新商品コーナー

**対象**: PLAZA、ロフト、アミング等のバラエティショップ（VS）

#### 狙いと理由

- **チャネルとの親和性**

  - 2Snackブランドの主戦場であるVS先は、パッケージデザインと素材のこだわりを最も評価してもらえるチャネル。ベーシックラインとの差別化（原料品質・乳酸菌K-1配合）が直接的な訴求力になる。
- **「価格の壁」を越えるため**

  - 定番棚で100円台の菓子と並ぶと「高い」と感じられるが、レジ前やエンドは「ついで買い・ご褒美」の心理が働きやすく、400円でもトライアル購入されやすい。
- **形状の最大活用**

  - 「フック穴」と「小ぶりなサイズ」は吊り下げ什器に最適。場所を取らずに導入できる点をバイヤーへの交渉材料にする。

#### 懸念点・リスク

- **競争率とコスト**

  - 最も売れる場所であるため、大手メーカーが年間契約で枠を押さえている場合や、協賛金等のコストが発生するケースがある。
- **回転率のシビアさ**

  - 「売れなければ即撤去」の場所であるため、初動の実績が重要。

### 2. サブ戦場：菓子売場の定番棚（小売先の棚構成に応じて柔軟に対応）

Phase 1では機能性表示がないため、「健康チョコ棚」一辺倒ではなく、**各小売先の棚構成に合わせた最適な定番棚**を提案する。

#### 棚パターン別の提案方針

| パターン | 対象棚                                                  | 想定される小売先                    | 提案の切り口                                                                                      |
| :------: | ------------------------------------------------------- | ----------------------------------- | ------------------------------------------------------------------------------------------------- |
|    A    | **素材こだわり系・プレミアムスナック / 輸入菓子** | PLAZAなどVS、高級SM（北野エース等） | 原料のこだわり・パッケージデザインの世界観で訴求。輸入菓子と並んでも価格帯・質感が合う            |
|    B    | **健康志向チョコレート棚**                        | 成城石井、ナチュラルローソン等      | 乳酸菌K-1配合マークで健康軸の訴求。チョコレート効果・LIBERA等と並ぶ「大人の健康おやつ」ポジション |
|    C    | **米菓・おせんべいコーナー**                      | SM系（ヤオコー、ライフ等）          | 抹茶塩おかきは米菓に近い商品特性。型物コーナーへの展開で「プレミアム米菓」という新しいポジション  |

#### 棚パターン共通の強み

- **白パッケージによる差別化（視認性）**
  - どの棚でも、競合品は「茶・黒・赤・濃緑」など濃い色が中心。マットな白は棚の中で圧倒的に目立つ（カラーブロッキング効果）。

#### 懸念点・リスク

- **NB商品との価格差**

  - どの棚でも競合NB商品（200円〜300円）と比べ400円は割高に映る。素材のこだわりとブランドストーリーでの価値訴求が不可欠。
- **味の誤認リスク（パターンA・B）**

  - 白パッケージが「ホワイトチョコ」と勘違いされる可能性。POP等で補足が必要。
- **カテゴリ認識の混乱（パターンC）**

  - おかき棚に置かれた場合、チョコレートラスクSKUとの統一展開が難しくなる可能性がある。

### 3. プラスアルファの提案：シリアル・グラノーラ売場（サイドネット等）

#### 狙いと理由

- **素材の親和性とクロスMD**

  - 「穀物（ラスク・おかき）× チョコ」という製品特性は、グラノーラ好きの嗜好と合致する。
- **新しい食べ方の提案**

  - 「ヨーグルトへのトッピング」や「朝食のついでに買う午後のご褒美」として提案することで、菓子売場に立ち寄らない層へもリーチできる。

#### 懸念点・リスク

- **担当バイヤーの縦割り**

  - 企業によってはシリアルが「食品部門（グロサリー）」扱いとなり、菓子バイヤーの管轄外で提案が通らない場合がある。
- **露出の弱さ**

  - 菓子売場に比べて来店頻度・立ち寄り率が低いため、単独で置くと回転が悪くなる恐れがある。

## Phase 2（2027年4月〜）: 機能性表示取得後の展開変化

### 訴求の強化ポイント

機能性表示食品マーク + ヘルスクレーム（肌のうるおい・整腸）がパッケージに追加されることで、**商品の信用度が上がり、提案の説得力が大きく向上する。**

| 項目               | Phase 1                         | Phase 2                                                          |
| ------------------ | ------------------------------- | ---------------------------------------------------------------- |
| 訴求できること     | 「乳酸菌K-1配合」（事実の記載） | 「肌のうるおいを守る」「おなかの調子を整える」（ヘルスクレーム） |
| バイヤーへの提案力 | 素材こだわり・デザイン中心      | 健康エビデンスを武器にした棚提案が可能                           |
| 消費者への訴求力   | ブランド・パッケージで選ばれる  | 機能性への信頼感で選ばれる（リピート率向上）                     |

### 棚展開の変化（小売先の棚構成に応じて柔軟に対応）

| 小売先の棚構成                             | Phase 2での対応                                                    |
| ------------------------------------------ | ------------------------------------------------------------------ |
| **機能性表示食品コーナー**がある店舗 | コーナーへの移動を提案。機能性表示食品としての視認性UP             |
| **健康菓子コーナー**がある店舗       | 健康菓子コーナーへの移動を提案。ヘルスクレームが棚との整合性を強化 |
| 上記がない店舗                             | Phase 1と同じ棚を維持し、POP・什器の訴求をヘルスクレーム入りに更新 |

### 新規チャネルの可能性

- **ドラッグストア**: 機能性表示を取ることで「格上げ」になる可能性がある。ただし、売価400円前後はドラッグストアの菓子棚では厳しい価格帯であり、導入可否は慎重に判断する。
- **あくまでバラエティショップでしっかり回せる商品にすることが最優先。** ドラッグストアはポテンシャル枠として位置づける。

## Phase 1 → 2 の移行について

### ロット構造の前提

| 項目         | 数量             | 備考                         |
| ------------ | ---------------- | ---------------------------- |
| 製造ロット   | 15,000pc / 4か月 | 1SKUあたり                   |
| 包材ロット   | 27,000pc         | 1SKUあたり。製造ロットとは別 |
| 包材 ÷ 製造 | 約1.8回分        | 包材1発注で製造約1.8ロット分 |

### パッケージ切替のシナリオ

Phase 1パッケージ（機能性表示なし）を包材ロット最低の27,000pc発注した場合：

| 時期                                    | 製造         | パッケージ使用状況                                          |
| --------------------------------------- | ------------ | ----------------------------------------------------------- |
| **1stロット**（2026年9月〜12月）  | 15,000pc製造 | Phase 1パッケージ 15,000pc使用 →**残12,000pc**       |
| **2ndロット**（2027年1月〜4月）   | 15,000pc製造 | Phase 1パッケージ残12,000pc使用 →**3,000pc分が不足** |
| **機能性表示取得**（2027年4月〜） | —           | Phase 2パッケージ（機能性表示あり）の発注が可能に           |

#### 想定される対応パターン

|  パターン  | 内容                                                                | メリット                                          | デメリット                                       |
| :---------: | ------------------------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------ |
| **A** | 2ndロットの不足3,000pc分からPhase 2パッケージ（27,000pc発注）に切替 | 在庫ロスなし。Phase 2パッケージの立ち上がりが早い | 2ndロット内にPhase 1/2が混在する                 |
| **B** | 2ndロットはPhase 1パッケージ12,000pcのみ製造し、以降をPhase 2に切替 | Phase区切りが明確                                 | 2ndロットが製造ロット割れ（12,000pc < 15,000pc） |
| **C** | Phase 1パッケージの在庫をPR・販促で早期消化し、Phase 2に一括切替    | ブランドメッセージが統一しやすい                  | 販促コストが発生。消化が計画通りいかないリスク   |

> 最適なパターンは、2ndロット製造判断時点（2027年1月頃）の販売実績・在庫状況・機能性表示の申請進捗によって決定する。
