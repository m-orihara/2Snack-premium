# 発売シナリオ分析 — 主要企業の採用パターンと判断基準

**作成日**: 2026-03-02
**参照元**: [第2販売戦略.md](第2販売戦略.md)、[プレミアムライン販売計画.md](プレミアムライン販売計画.md)

---

## 1. 目的

前回チームMTG（2/28、折原・小松・室屋）にて、「主要取引先ごとの採用パターンによる影響を分析し、パターン別の判断基準を整理する」という方針が確認された。

本資料では以下を明らかにする:

1. 主要3社（PLAZA・NL・カフェランテ）の個別貢献度
2. 採用パターン別のロット判定（7パターン）
3. 各パターンにおける推奨アクション
4. スケジュール上の判断ポイントと判断フロー

> **PLAZA + NLの2社だけでリアルケースの約44%を占める。** 全35社中わずか2社が全体の半分近くを支えており、この2社の採用可否がロット判定を決定的に左右する。

### 2-2. PLAZA先行販売の追加効果

PLAZA先行時は日販が0.2→0.4に上昇（代わりに他社は14日間の販売期間短縮）。

| 指標         | PLAZA先行あり | PLAZA先行なし |       差分       |
| ------------ | :-----------: | :-----------: | :--------------: |
| 含浸チョコ   |    15,210    |    13,576    | **+1,634** |
| ケーキラスク |    10,753    |     9,777     |  **+976**  |

> PLAZA先行販売の有無で含浸チョコは約1,600個、ケーキラスクは約1,000個の差が出る。特に含浸チョコはこの+1,634がロットクリアの分水嶺（先行ありで+210、先行なしで▲1,424）。

## 3. シナリオパターン（7パターン）

### 3-1. パターン定義

| #           | パターン名                   | PLAZA | NL | カフェ | 発生イメージ                          |
| ----------- | ---------------------------- | :----: | :-: | :----: | ------------------------------------- |
| **A** | **全順調**             | ○先行 | ○ |   ○   | 全社順調に採用。PLAZA先行販売も成立   |
| **B** | **PLAZA先行なし**      | ○通常 | ○ |   ○   | 全社採用するが、PLAZA先行販売は不成立 |
| **C** | **NL失注（先行あり）** | ○先行 | × |   ○   | NLのみ不採用。PLAZA先行は成立         |
| **D** | **NL失注（先行なし）** | ○通常 | × |   ○   | NL不採用、かつPLAZA先行も不成立       |
| **E** | **PLAZA失注**          |   ×   | ○ |   ○   | PLAZAが不採用（NLは採用）             |
| **F** | **PLAZA・NL両失注**    |   ×   | × |   ○   | 主要2社とも不採用                     |
| **G** | **主要3社全失注**      |   ×   | × |   ×   | 最悪ケース                            |

> カフェランテは構成比6.6〜6.9%と影響が限定的なため、パターンF/G以外は「採用」として扱う。

### 3-2. 判定サマリー

**前提:**

- 含浸チョコ: Cap×1.0、ロット15,000pc/4か月、EC=+2,500pc(「含浸チョコ卸数量」に2,500㍶を足している)
- ケーキラスク: Cap×1/2（2SKU）、ロット基準=チョコレート4,465pc/3か月（最も厳しいSKU）

| #           | パターン           | 含浸チョコ 卸数量 | 含浸チョコ ロット差(卸のみ) | 含浸チョコ ロット差(卸+EC) | ケーキラスク ×1/2 | ケーキラスク ロット差(Choco基準) | **総合判定**           |
| ----------- | ------------------ | ----------------: | :-------------------------: | :------------------------: | -----------------: | :------------------------------: | ---------------------------- |
| **A** | 全順調（先行あり） |            15,210 |       **+210**       |           +2,710           |              5,377 |          **+912**          | **両商品GO**           |
| **B** | PLAZA先行なし      |            13,576 |           ▲1,424           |      **+1,076**      |              4,889 |          **+424**          | **両商品GO（EC必須）** |
| **C** | NL失注（先行あり） |            12,454 |           ▲2,546           |       **▲46**       |              4,389 |          **▲76**          | **ボーダーライン**     |
| **D** | NL失注（先行なし） |            10,456 |           ▲4,544           |          ▲2,044          |              3,719 |              ▲746              | **発売見送り検討**     |
| **E** | PLAZA失注          |            10,696 |           ▲4,304           |          ▲1,804          |              3,809 |              ▲656              | **発売見送り検討**     |
| **F** | PLAZA・NL両失注    |             7,576 |           ▲7,424           |          ▲4,924          |              2,639 |             ▲1,826             | **発売見送り**         |
| **G** | 主要3社全失注      |             6,676 |           ▲8,324           |          ▲5,824          |              2,301 |             ▲2,164             | **発売見送り**         |

> **読み取れる構造:** パターンA〜Cが「発売可能圏」、D以降が「発売困難圏」。境界線はパターンC（NL失注+PLAZA先行あり）にある。

## 4. パターン別の詳細と推奨アクション

### パターンA: 全順調（PLAZA先行あり）— 両商品GO

| 商品                        |           数量 | vs ロット | 判定 |
| --------------------------- | -------------: | --------: | :--: |
| 含浸チョコ（卸のみ）        |         15,210 |      +210 |  ✓  |
| 含浸チョコ（卸+EC）         |         17,710 |    +2,710 |  ✓  |
| ケーキラスク CK（×1/2）    | 5,377 vs 3,974 |    +1,403 |  ✓  |
| ケーキラスク Choco（×1/2） | 5,377 vs 4,465 |      +912 |  ✓  |
| ケーキラスク 抹茶（×1/2）  | 5,377 vs 4,140 |    +1,237 |  ✓  |

**推奨アクション:** 予定通り9月に両商品同時発売。PLAZA先行販売を8月中旬に実施。

**留意点:** 含浸チョコの卸のみでのマージンは+210と薄い。ECを並行で立ち上げ、+2,710の余裕を確保しておくことが望ましい。

---

### パターンB: PLAZA先行なし — 両商品GO（含浸チョコはEC必須）

| 商品                        |   数量 |  vs ロット | 判定 |
| --------------------------- | -----: | ---------: | :--: |
| 含浸チョコ（卸のみ）        | 13,576 |    ▲1,424 |  ✗  |
| 含浸チョコ（卸+EC）         | 16,076 |     +1,076 |  ✓  |
| ケーキラスク 全SKU（×1/2） |  4,889 | +424〜+915 | 全✓ |

**推奨アクション:** 9月に両商品発売。ただし含浸チョコは**EC販売（月625個）を確実に実施する前提**。EC販売体制の構築を5月までに並行で進める。

**リスク:** 含浸チョコの卸のみでは▲1,424のロット割れ。EC販売の遅延や不振は即ロット余り（約22万円のロス）に直結する。

---

### パターンC: NL失注 + PLAZA先行あり — ボーダーライン

| 商品                        |           数量 |      vs ロット |        判定        |
| --------------------------- | -------------: | -------------: | :----------------: |
| 含浸チョコ（卸のみ）        |         12,454 |        ▲2,546 |         ✗         |
| 含浸チョコ（卸+EC）         |         14,954 | **▲46** | ✗（ほぼボーダー） |
| ケーキラスク CK（×1/2）    | 4,389 vs 3,974 | **+415** |         ✓         |
| ケーキラスク Choco（×1/2） | 4,389 vs 4,465 | **▲76** |         ✗         |
| ケーキラスク 抹茶（×1/2）  | 4,389 vs 4,140 | **+249** |         ✓         |

**推奨アクション:**

- **ケーキラスク**: SKU選定で**CK + 抹茶を選べばGO可能**（CK +415, 抹茶 +249）。チョコレートSKUはロット4,465に対し▲76のため回避する
- **含浸チョコ**: EC込みでも▲46。以下のいずれかで対応:
  - 確度「無」企業への営業強化（リアル値のある11社のうち1社でも獲得すればカバー可能）
  - ECの上振れ期待（月625個が月629個でカバー。+4個/月）
- **判断**: 上記の補完が見込めればケーキラスク発売 + 含浸チョコも発売。見込めなければケーキラスクのみ先行発売

**このパターンの重要性:** NL失注でもPLAZA先行が成立すれば、ほぼ発売可能圏内。**SKU選定（Choco回避）とEC/リスト拡充の二つが鍵**。

> **参考: SKU選定への示唆**
> パターンCを想定すると、2SKUの選定は CK + 抹茶 が最も安全。ただしChoco は営業利益が最も高い（84.0円 vs CK 63.0円 / 抹茶 70.8円）ため、利益面でのトレードオフがある。パターンA/Bではどの組み合わせでもクリアするため、NL失注リスクをどこまで織り込むかの判断になる。

| 商品                        |           数量 | vs ロット | 判定 |

| 含浸チョコ（卸+EC）         |         12,956 |   ▲2,044 |  ✗  |
| ケーキラスク CK（×1/2）    | 3,719 vs 3,974 |     ▲255 |  ✗  |
| ケーキラスク Choco（×1/2） | 3,719 vs 4,465 |     ▲746 |  ✗  |
| ケーキラスク 抹茶（×1/2）  | 3,719 vs 4,140 |     ▲421 |  ✗  |

**ロス金額:** 含浸チョコ ▲312,139円（EC込み）

**推奨アクション:**

### パターンE: PLAZA失注 — 発売見送り検討

| 商品                        |           数量 | vs ロット | 判定 |
| --------------------------- | -------------: | --------: | :--: |
| 含浸チョコ（卸+EC）         |         13,196 |   ▲1,804 |  ✗  |
| ケーキラスク CK（×1/2）    | 3,809 vs 3,974 |     ▲165 |  ✗  |
| ケーキラスク Choco（×1/2） | 3,809 vs 4,465 |     ▲656 |  ✗  |
| ケーキラスク 抹茶（×1/2）  | 3,809 vs 4,140 |     ▲331 |  ✗  |

**ロス金額:** 含浸チョコ ▲275,489円（EC込み）

**推奨アクション:**

- **9月発売見送り**
- PLAZAの不採用理由を分析し、再提案の余地を探る
- アタックリスト大幅拡充（確度「無」のリアル値あり11社への本格営業）
- 発売時期を後ろ倒し（半年〜1年）、PLAZAに再アプローチ

**このパターンの意味:** PLAZAの失注は (1) 直接の数量減（含浸チョコ ▲2,880 / ケーキラスク ▲2,160）に加え、(2) 先行販売の効果（含浸チョコ +1,634 / ケーキラスク +976）も同時に失う。**PLAZAの失注 = 事実上の発売見送りトリガー。**

  - アタックリストの抜本的拡充

## 5. 判断フロー

### 5-1. スケジュール上の判断ポイント

ガントチャート（プレミアムライン販売計画.md）と照合した意思決定タイミング:

| 時期              | イベント                         | 判断内容                                 | 必要な情報                                     |
| ----------------- | -------------------------------- | ---------------------------------------- | ---------------------------------------------- |
| **3月前半** | デザイン制作中、売価議論開始     | シナリオをチーム共有。社長報告に組み込む | 本資料の完成                                   |
| **4/3**     | **デザインFIX**            | **SKU2種の確定**                   | パターンC対応を見据えた選定（CK+抹茶が安全側） |
| **4月後半** | アポ取り                         | PLAZA/NLの採用可否の初期感触を取得       | 各社バイヤーの反応                             |
| **5月前半** | **商品発注・営業期間開始** | **GO/NO-GO最終判断**               | PLAZA/NLの採用確度                             |
| **7月前半** | 受注締め                         | 受注数量の確認。パターンB/Cへの移行判断  | 全社の受注状況                                 |
| **8月中旬** | PLAZA先行販売                    | 先行実施の最終判断                       | PLAZA側の準備状況                              |

### 5-2. 判断フローチャート

**5月前半 GO/NO-GO判断時点:**

```
PLAZA 採用確定？
│
├── YES
│   │
│   ├── NL 採用確定？
│   │   │
│   │   ├── YES ─── PLAZA先行OK？
│   │   │            ├── YES → パターンA → 両商品 9月発売
│   │   │            └── NO  → パターンB → 両商品 9月発売（EC必須）
│   │   │
│   │   └── NO ──── PLAZA先行OK？
│   │                ├── YES → パターンC → ケーキラスク発売（SKU次第）
│   │                │                      + 含浸チョコはリスト拡充で+46個補完
│   │                └── NO  → パターンD → 9月発売見送り
│   │
│   └── (NL未確定) → パターンA/B想定で準備を進めつつ、NL確定を待つ
│
└── NO → パターンE〜G → 9月発売見送り
```

### 5-3. 発売可否の最低条件

| 条件                 | 含浸チョコ                          | ケーキラスク                  |
| -------------------- | ----------------------------------- | ----------------------------- |
| **最低条件**   | PLAZA採用 + EC販売体制              | PLAZA採用                     |
| **安全ライン** | PLAZA先行 + NL採用                  | PLAZA + NL採用                |
| **なくても可** | NL（PLAZA先行+EC+リスト拡充で補完） | NL（PLAZA先行+SKU選定で補完） |

> **結論: PLAZAの採用が両商品の発売における必須条件。** NLは重要だが、PLAZA先行販売とEC/リスト拡充で補完の余地がある。
