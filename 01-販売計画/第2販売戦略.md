# 第2販売戦略 — 採用枠制限（Cap）シミュレーション

**作成日**: 2026-02-20
**最終更新**: 2026-03-02
**参照元**: [プレミアムライン販売計画.md](プレミアムライン販売計画.md)

---

## 1. 背景・目的

元販売計画（プレミアムライン販売計画.md）では、全取引先が全SKUを採用する前提で試算を行っている。しかし、プレミアムライン全5SKUは「プレミアムチョコ系菓子」という同一カテゴリーに属するため、取引先の棚構成上、全SKUが同時に採用されることは現実的でない。

本資料では、**1社あたり最大3SKU（Cap=3）** という採用枠制限を設け、その制約下で各商品が製造ロットを消化できるかを検証する。

---

## 2. 商品構成（全5SKU）

| 商品           | SKU                                      |    SKU数    |
| -------------- | ---------------------------------------- | :---------: |
| がんしんチョコ | オレンジチョコレートラスク、抹茶塩おかき |      2      |
| ケーキラスク   | チーズケーキ、チョコレート、抹茶         |      3      |
| **合計** |                                          | **5** |

---

## 3. 前提条件

### 3-1. 出発点：なぜ採用枠制限を考えるのか

元販売計画では「全取引先が全SKUを採用する」前提だが、現実には以下の制約がある。

- 5SKUすべてが同一カテゴリー（プレミアムチョコ系菓子）
- 取引先の棚スペースには限りがあり、同カテゴリーで5SKUすべてを並べるのは非現実的
- バイヤーは限られた棚枠の中から、自店舗に合ったSKUを取捨選択する

**→ 1社あたりの採用上限（Cap）を設定し、その影響をシミュレーションする必要がある**

### 3-2. Cap = 3SKU の設定根拠

| 項目                                 | 値              |
| ------------------------------------ | --------------- |
| 全SKU数                              | 5 SKU           |
| **1社あたり最大採用枠（Cap）** | **3 SKU** |

### 3-3. 仮説：バイヤーはCap 3枠をどう使うか

#### がんしんチョコ（2SKU）→ 2SKUとも確実に採用される

**→ 係数 1.0（掛け目なし）= 元販売計画と同じ数量で試算**

#### ケーキラスク（3SKU）→ 3SKUのうち1SKUだけ採用される

**→ 係数 1/3（≈0.33）= ベース数量の3分の1で試算**

### 3-4. 係数の検算（Cap=3と整合するか）

| 商品           |    SKU数    | 係数 | 占有枠（係数 × SKU数） |
| -------------- | :---------: | :--: | :---------------------: |
| がんしんチョコ |      2      | 1.0 |      **2枠**      |
| ケーキラスク   |      3      | 1/3 |      **1枠**      |
| **合計** | **5** |  —  | **3枠 = Cap** ✓ |

### 3-5. 製造ロット

**がんしんチョコ**: 工場の最低ロット **15,000pc**（ロット期間: 4か月）

**ケーキラスク**: 工場の最低発注条件は**ラスク本体の総額 150万円**。

| SKU          | ラスク本体単価 | 最低ロット（3SKU・50万/SKU） | 最低ロット（2SKU・75万/SKU） | ロット期間 |
| ------------ | -------------: | :--------------------------: | :--------------------------: | :--------: |
| チーズケーキ |       188.75円 |           2,650pc           |           3,974pc           |   3か月   |
| チョコレート |       168.00円 |           2,977pc           |           4,465pc           |   3か月   |
| 抹茶         |       181.20円 |           2,760pc           |           4,140pc           |   3か月   |

## 9. 結論

### 総括

| 項目             |         がんしんチョコ         |   ケーキラスク（3SKU）   |
| ---------------- | :-----------------------------: | :-----------------------: |
| 最低ロット       |        15,000pc / 4か月        |  2,650〜2,977pc / 3か月  |
| 係数             |        1.0（掛け目なし）        |            1/3            |
| ロットクリア     | ポジティブ・リアル+PLAZA（2/4） | コンサバ以外クリア（3/4） |
| リスク評価       |        **中〜高**        |     **低〜中**     |

## 10. ケーキラスク 2SKU展開シナリオ

### 10-1. 検討背景

2026-02-28 チーム内MTG（折原・小松・室屋）にて、ケーキラスクの9月発売時SKU数について議論を実施。

**チーム方針**: ケーキラスクは**2SKUでの発売を基本方針**とし、3SKU目は売れ行きを見て追加を検討する。

### 10-5. 3SKU vs 2SKU 比較

| 指標                   | 3SKU（×1/3・ロット2,650〜2,977） | 2SKU（×1/2・ロット3,974〜4,465） |
| ---------------------- | :-------------------------------: | :-------------------------------: |
| リアル+PLAZAのロット差 |            +607〜+934            |           +912〜+1,403           |
| リアルのロット差       |            +282〜+609            |            +424〜+915            |
| コンサバのロット差     |           ▲647〜▲974           |          ▲969〜▲1,460          |
| ロットクリア判定       |          3/4ケースクリア          |          3/4ケースクリア          |
| 管理するSKU数          |                3種                |                2種                |

**評価**: 2SKUが優位。

## 11. リスク分析と対応策

### 11-3. リスクの重要度と対応策

| # | リスク                                                                | 重要度 | 発生可能性 | 対応策                                                                  |
| - | --------------------------------------------------------------------- | :----: | :--------: | ----------------------------------------------------------------------- |
| 1 | **PLAZAが採用しない**                                           |  極高  |     中     | 先行販売の提案で差別化。不採用確定時は発売時期を後ろ倒し検討            |
| 2 | **含浸チョコのロット消化不足**（構造的課題）                    |   高   |     高     | EC上積み、アタックリスト拡充、PLAZA先行販売の実施                       |
| 3 | **NLが採用しない**                                              |   高   |     中     | 確度「無」企業への積極営業で補完                                        |

### 11-4. 意思決定のトリガー

| 条件                                   | アクション                                 |
| -------------------------------------- | ------------------------------------------ |
| PLAZAの採用が確定しない                | 発売時期の後ろ倒し（半年〜）を検討         |
| PLAZAは確定、他社が不調                | 予定通り9月発売。EC強化で補完              |
| 含浸チョコのロット消化見通しが立たない | ケーキラスクのみ先行発売を検討             |
| ケーキラスク2SKUの売れ行きが好調       | 3SKU目（保留フレーバー）の追加を工場と調整 |
